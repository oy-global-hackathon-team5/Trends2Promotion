# 올리브영 글로벌 GenAI 해커톤 Team 5

## 개요
### AI 기반 트렌드 키워드 프로모션 자동화
1. Google Trends의 실시간 트렌드 키워드 수집
2. 수집된 키워드를 통한 현상황 트렌드 파악
3. 트렌드와 올리브영 상품 데이터 매핑
4. 올리브영 상품 데이터 이미지를 활용한 프로모션 생성

### GenAI 기술 사용
> **사용 스택**

- Next.js (FullStack Framework)
- Supabase (Database)
- Gemini 2.5 pro (Thinking AI Model)
- Gemini 2.5 flash image (NanoBanana) (Image Generation)
- Vertex AI Studio (System Prompt Compare and Test)
- Vercel (Deploy)
