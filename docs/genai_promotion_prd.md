# GenAI 기반 국가별 트렌드 자동 기획전 생성 시스템  
**Product Requirements Document (PRD)**

---

## 1. 프로젝트 개요

글로벌 뷰티·코스메틱 커머스 서비스에서는 국가별 문화, 취향, 실시간 검색 트렌드가 크게 다릅니다.  
기존에는 트렌드 조사 → 상품 선정 → 이미지 제작 → 기획전 구성까지 모두 수작업으로 진행되어 운영 리소스가 많이 소요되었습니다.

본 프로젝트는 **GenAI를 활용하여 국가별 트렌드 수집 → 상품 매핑 → 이미지 생성 → 기획전 구성까지 자동화되는 시스템**을 구축하는 것을 목표로 합니다.

---

## 2. 왜 필요한가 (문제 & 기대 가치)

### 문제점
- 국가별 기획전 생성에 MD/디자이너가 많은 수작업 투입  
- 기획전·배너 이미지 제작 비용이 국가마다 발생  
- 국가별 실시간 트렌드 반영이 느리고 수동적  
- 글로벌 확장 시 국가 증가에 비례해 운영 비용 증가  

### 기대 가치
- **GenAI가 자동으로 기획전을 생성**하여 운영 리소스 대폭 절감  
- **이미지 생성까지 AI가 자동 처리**, 외주·디자인 비용 감소  
- **국가별 인기 트렌드를 실시간 반영한 현지화**가 가능  
- 국가 추가 시 동일 리소스로 확장 가능한 구조 확보  

---

## 3. 핵심 기능 (High-Level Features)

### 3.1 국가별 트렌드 자동 수집
- Google Trends에서 국가별 인기 검색어 데이터를 가져옴  
- 뷰티·코스메틱 분야 키워드를 필터링하는 로직 포함
- 추후 Cron 등 자동화 고려할 수 있음

### 3.2 GenAI 기반 키워드 분석 & 상품 매핑
- GenAI가 의미 기반으로 뷰티 연관 키워드를 추출
- 당사 상품 메타데이터와 연관도를 분석해 상품 세트 자동 구성  
- 각 키워드에 최적화된 n개의 상품 그룹 추천  

### 3.3 GenAI 기반 이미지 생성
- 기획전 테마에 따라 AI 이미지 모델(Nano Banana 등)을 이용해 자동 생성  
  - 히어로 배너 이미지  
  - 기획전 상세 페이지 이미지
- 국가별 스타일·톤을 프롬프트로 반영 가능  

### 3.4 기획전 데이터 저장
- 생성된 기획전 정보(키워드, 상품 세트, 이미지 URL 등)를 DB에 저장

### 3.5 프론트엔드 반영
- 사용자가 웹/앱에서 국가 변경 시  
  → DB에 저장된 해당 국가 기획전을 자동으로 호출하여 노출  
- 히어로 배너 클릭 시 자동 생성된 기획전 상세페이지로 이동  

---

## 4. 기대 효과 요약

- **운영 자동화**: 기획전 생성의 대부분을 AI가 처리  
- **비용 절감**: 이미지 제작·외주 비용 최소화  
- **현지화 강화**: 국가별 실시간 인기 트렌드 기반 구성  
- **확장 용이성**: 국가가 늘어나도 추가 리소스 없이 운영 가능
